reclass (line 639)

Called on a TPObject, return an instance of the appropriate subclass

based on the object's type. This is necessary in cases when a batched subrequest returns an object whose type is not known until the batch is run--for example, an endpoint that returns a TPAsset which may turn out to be a TPPost, TPComment, etc.
